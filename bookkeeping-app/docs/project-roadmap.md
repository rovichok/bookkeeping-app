```md
# Project Roadmap

This roadmap shows the current phase of Lentis and the immediate next stages. It is designed to support growth without disturbing the approved Step 1 baseline.

## Step 1 — Approved Baseline

Status: Complete and frozen

Step 1 establishes the foundation of the project:

- approved design direction
- approved current content and information
- approved source code baseline
- initial visual and structural identity

This step acts as the protected reference point for future work.

## Pre-Step 2 / Pre-Step 3 Work

Status: Allowed now

Before moving to Step 2 and Step 3, only the following work is intended:

- branch strategy setup
- repository documentation
- GitHub workflow preparation
- subtle UI polish that does not alter design or content

This stage is intentionally non-destructive.

## Step 2 — Service Page Expansion

Planned goals:

- add service-focused pages
- structure the app for clearer visitor journeys
- strengthen conversion intent without redesigning the brand
- support money pages such as cleanup and monthly bookkeeping

Likely focus areas:

- Bookkeeping Cleanup
- Monthly Bookkeeping
- QuickBooks Support

## Step 3 — Conversion Content and Positioning

Planned goals:

- refine homepage messaging for stronger conversion
- strengthen trust signals
- improve service positioning
- support lead generation more directly

Possible additions:

- conversion-focused headlines
- stronger CTA structure
- FAQ content
- credibility blocks
- improved lead paths

## Future Work After Step 3

Planned areas beyond the current scope:

- SEO foundation
- CI/CD setup
- hosting and deployment strategy
- analytics
- form handling / lead pipeline
- technical performance optimization
```
